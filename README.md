## 基础环境
本人使用 Ubuntu22.04 基础工程环境，vscode 阅读代码，Linux V6.6 版本。
所有内容以代码及官方文档为准！！！
```
sudo apt update && \
sudo apt build-dep linux && \
sudo apt-get install libncurses-dev qemu qemu-system gdb-multiarch gcc-aarch64-linux-gnu bear global clangd flex
```
